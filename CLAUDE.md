# プロジェクト固有ルール

## 実装ルール

### 1. スコープ厳守
- 指示されたタスク（Phase/Work）以上のことは絶対に実行しない
- 指示されたPhaseが完了したら必ず作業を停止する
- 完了後は何をどのように実装したかを報告する

### 2. 報告形式
Phase完了時に以下を報告：
- 実装したファイル一覧
- 各ファイルの概要
- 動作確認方法
- 次のPhaseに進む準備ができているか

### 3. 進捗管理
- タスク完了時は progress.txt を更新
- 完了タスクは progress-archive.txt に移動

## 技術スタック
- React（初学者向けコード）
- JavaScript
- GitHub Pages でホスティング
